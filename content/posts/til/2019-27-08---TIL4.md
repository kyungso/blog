## 리듀서

상태를 변화시키는 로직이 있는 함수

예제 코드

``` JavaScript
(...)
const initialState = {
  number: 1,
  foo: 'bar',
  baz: 'qux'
};

function counter(state = initialState, action) {
  switch (action.type) {
    case INCREMENT:
      return Object.assign({}, state, {
        number: state.number + action.diff
      });
    case DECREMENT:
      return Object.assign({}, state, {
        number: state.number - action.diff
      });
    default:
      return state;
  }
}
```

<br>

 => Object.assign 대신 ES6 문법의 전개 연산자(...)를 사용하면 더욱 깔끔하게 코드를 입력할 수 있습니다.

``` JavaScript
function counter(state = initialState, action) {
  switch(action.type) {
    case INCREMENT:
      return {
        ...state,
        number: state.number + action.diff
      };
    case DECREMENT:
      return {
        ...state,
        number: state.number - action.diff
      };
    default:
      return state;
  }
}
```
